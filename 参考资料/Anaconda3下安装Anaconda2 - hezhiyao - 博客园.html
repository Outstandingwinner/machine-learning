Anaconda3下安装Anaconda2

一、下载软件

下载Anaconda Python3.6版本和Anaconda Python2.7版本。下载地址：https://www.anaconda.com/download/   

 下载需要FQ，也可以从其他源下载。

 

二、安装

1、先安装Anaconda Python3.6，注意勾选下面两个选项。

打钩的两个选项的大概意思是： 

  • Add Anaconda to the system PATH environment variable 

这个选项是将安装目录加入到系统的PATH环境变量中，以后在命令行中可以直接用python命令启动python解释器。 

  • Register Anaconda as the system Python2.7

这个选项是让其他IDE能够检测到python3.6并将设置为默认的Python3.6。

 

2、我们安装好Anaconda Python3.6之后，再来安装Anaconda Python2.7。需要注意两点：

  • 此时Anaconda Python2.7的安装目录必须选在Anaconda3\envs子目录下，例如我的目录是C:\Users\hezhiyao\Anaconda3\envs\anaconda2
  • 注意安装时要取消勾选下图这两个选项，注意要取消。

 

 三、启动python

1、打开windows终端,输入python,默认打开Anaconda Python3.6,因为安装时勾选了那两项。

 

2、如果想启动python2.7,则首先输入命令  activate anaconda2 , 激活anaconda2, 然后输入 python 即可。
